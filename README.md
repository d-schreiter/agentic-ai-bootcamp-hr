# 🤖 watsonx Agentic AI Bootcamp

![alt text](/agentic-bootcamp.png)


Welcome to the watsonx Agentic AI Bootcamp! This hands-on learning experience will guide you through building and deploying enterprise-grade Agentic AI applications using IBM watsonx.

## 🎯 What You'll Learn

In this bootcamp, you will:

🚀 **Understand Agentic AI Concepts** - Learn the fundamental principles behind autonomous AI agents and multi-agent orchestration

🚀 **Build with watsonx** - Leverage IBM watsonx and open source tools for rapid, low-code prototyping of Agentic AI workflows

🚀 **Deploy Production-Ready Solutions** - Learn deployment strategies and trust considerations for taking AI agents to production

🚀 **Solve Real Business Problems** - Work through enterprise use cases with step-by-step guidance

## ⏱️ Bootcamp Structure

Each lab is designed to be completed in **approximately 1.5 hours** and includes:
- 📖 Business context and problem statement
- 🏗️ Architecture overview
- 🎥 Demo video
- 📝 Step-by-step hands-on instructions
- 💡 Key learning outcomes

## 🚀 Getting Started

### Prerequisites

Before starting the labs, ensure you have:
- Access to IBM watsonx Orchestrate
- Basic understanding of AI/ML concepts (helpful but not required)
- A web browser and internet connection

### How to Use This Repository

1. **Choose a use case** from the table below that aligns with your interests
2. **Navigate to the use case folder** and read the README for context
3. **Follow the hands-on lab guide** step-by-step
4. **Experiment and customize** the solution to deepen your understanding

## 📚 Bootcamp Labs

Complete the labs in the following order for the best learning experience. Each lab builds upon concepts from the previous one:

### Lab 1: HR Automation (AskHR) - Foundation Lab
**Duration:** ~1.5 hours | **Start here** ⭐

Build an AI agent to automate routine HR tasks like checking leave balance, requesting time off, and updating employee details. Solve TechCorp's challenge of managing HR operations for 100,000 employees.

**What you'll learn:**
- Core Agentic AI concepts and multi-agent orchestration
- RAG (Retrieval-Augmented Generation) fundamentals
- Backend system integration with OpenAPI
- No-code agent development in watsonx Orchestrate

**[👉 Start Lab 1: HR Automation](./usecases/ask-hr)**

---

### Lab 2: HR Talent Management - Advanced Lab
**Duration:** ~1.5 hours | **Complete after Lab 1**

Help Luisa, an HR manager, automate the recruiting process for 5,000 new hires. Scan résumés, match candidates to positions, coordinate interviews, and review Business Conduct Guidelines violations.

**What you'll learn:**
- Advanced document processing and analysis
- Complex multi-step workflow orchestration
- Chat with documents capabilities
- Intelligent document processing with flows

**[👉 Start Lab 2: HR Talent Management](./usecases/hr-talent)**

## 🎓 Learning Path

Follow this structured path through the bootcamp:

1. **Start with Lab 1 (AskHR)** - Builds foundational understanding of Agentic AI
2. **Progress to Lab 2 (HR Talent)** - Applies concepts to more complex scenarios
3. **Total Time:** ~3 hours for complete bootcamp experience

## 💡 Key Concepts Covered

Throughout these labs, you'll gain hands-on experience with:

- **Retrieval-Augmented Generation (RAG)** - Enhance AI responses with relevant document context
- **Multi-Agent Orchestration** - Coordinate multiple specialized agents to solve complex tasks
- **Tool Integration** - Connect AI agents to backend systems and APIs
- **No-Code Development** - Build sophisticated AI solutions without extensive coding
- **Intelligent Document Processing** - Extract and analyze information from unstructured documents
- **Workflow Automation** - Design and implement end-to-end business process automation
